# ASVAB Any Percent!
This is for practice on the most important parts of the ASVAB

Math and Verbs

## Installation

```bash
pip install asvab-anypct
```

This will install the ASVAB Any Percent! package and all of its dependencies.

## Usage

```bash
asvab-anypct
```

This will start the ASVAB Any Percent! application.

## Tests

There are two types of tests:
1. Math
2. Verbs

There are 3 diffrent levels of difficulty:
1. Easy
2. Medium
3. Hard

The more difficult the level, the more questions you will get.
